关于 sub2socks5

            

            

              基于 Go + sing-box 的本地代理管理器，把订阅、手动节点和节点组组织成可视化、可多端口分流的 SOCKS5 服务。

              支持 Docker 容器化部署、Web UI Bearer Token 鉴权、自动订阅更新、节点测速与一键配置。
